How Our AI Agents Think & Act

🧠 ReAct Pattern — Reasoning + Acting
Every AI decision in SkillForge goes through explicit Thought → Action → Observation loops before committing to a recommendation. This makes AI decisions transparent and auditable.
💭
Thought 1
Analyze the learner's situation
The agent reads the current context: day in plan, recent session scores, skill being studied. It forms a hypothesis about what action to take.
THOUGHT: The learner is on Day 8. Last 3 sessions averaged 52% — below the 70% mastery threshold.
I need to analyze performance patterns before deciding whether to continue or intervene.
⚡
Action 1
Call a tool to get real data
The agent calls a specific tool function with structured arguments. It waits for the result before continuing its reasoning chain.
ACTION: analyze_performance({"sessions": [last 3], "topic": "React Hooks"})
🔧 analyze_performance
👁️
Observation 1
Process tool result
The agent receives the tool's output and incorporates it into its understanding. The observation is concrete data — not a guess.
OBSERVATION: avgScore=52%, trend=declining (-8pts), topic="React Hooks".
Performance is below threshold. Weakness confirmed.
↻ Second Iteration — deeper investigation
💭
Thought 2
Plan the next targeted action
With data in hand, the agent reasons about what to do. Should it add review days? Check skill gaps more precisely?
THOUGHT: Score is 52% on React Hooks specifically. I should check skill gaps to see
if this is isolated or part of a broader JavaScript weakness before scheduling review.
⚡
Action 2
Check skill gaps precisely
The agent calls a second, more targeted tool to validate its hypothesis before making a plan change.
ACTION: check_skill_gaps({"skills": ["React Hooks", "useState", "useEffect", "async JS"]})
🔧 check_skill_gaps
👁️
Observation 2
Confirm the gap pattern
Real evidence confirms the weakness is isolated to hooks — not all JavaScript. The agent now has enough data to act decisively.
OBSERVATION: weakSkills=["React Hooks", "useEffect"], count=2.
Other JS skills are at 74%+. Targeted hook review needed.
✅
Final Answer
Transparent, evidence-backed recommendation
After 2 reasoning iterations with real tool data, the agent commits to a specific, justified action. The decision is fully auditable — every step is logged.
FINAL ANSWER: ✅ Adding 2 targeted review sessions for "React Hooks" (useEffect focus).
Avg score 52% with declining trend. Other JS skills healthy at 74%+.
Estimated 2 sessions to reach 70% mastery threshold.
Available Tools
analyze_performance()
Returns avg score, trend, topic from recent sessions
check_skill_gaps()
Finds skills below 70% mastery threshold

Returns job market demand and salary data for a skill
schedule_review()

🌐 9-Agent Network — How They Collaborate
Each learning journey activates all 9 specialized agents in sequence. They hand off context, debate adaptation decisions, and work in parallel where possible.
GoalAgent — Parses learning goal, detects domain
DecomposeAgent — Builds 4-6 skill dependency tree
DiagnosticAgent — Generates 5 MCQ knowledge check
ScoringAgent — Identifies gaps, calculates mastery %
CurriculumAgent — Builds 14-21 day personalized plan
EvaluatorAgent — Scores practice responses (4 criteria)
AdaptorAgent — Runs multi-agent debate, evolves plan
MarketAgent — Live job market & salary intelligence
SimulationAgent — Career scenario simulations
